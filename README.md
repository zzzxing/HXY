# 黄小游 V3.1（课堂云游平台）

这是一个面向信息技术课堂的桌面端云游学习平台，包含学生端、教师端、管理端，并内置“黄石国家矿山公园云游”完整案例。

## 小白启动（推荐）
### 第一次启动
1. 双击根目录 `start-first-time.bat`
2. 浏览器打开 `http://localhost:3000`

### 以后启动
1. 双击根目录 `start.bat`
2. 浏览器打开 `http://localhost:3000`

## 这版能直接演示什么
- 学生端：首页、路线/地图、点位探索、任务中心、证据背包、学习档案、班级共学
- 教师端：驾驶舱、路线配置、任务布置、课堂控制台、学生进度与档案评价
- 管理端：景区内容录入台、点位编辑器、资源模板库、学校/班级/用户管理

## 内置案例
- 主题：黄石国家矿山公园云游
- 点位：
  1. 序厅：黄石矿冶历史总览
  2. 露天矿坑观景台
  3. 矿车与采矿机械区
  4. 工业精神与工人故事展区
  5. 生态修复与城市转型展区

每个点位均内置：封面图、图集、视频、音频、观察点、任务、AI 上下文。
