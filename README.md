# 黄小游 V3.2（课堂云游平台）

黄小游 V3.2 是面向信息技术课堂的桌面端云游学习平台，包含：
- 学生端：云游学习与证据整理
- 教师端：案例工作台 + 课堂驾驶舱
- 管理员端：账号与班级管理台

## 一键启动（小白版）
### 首次启动
1. 双击 `start-first-time.bat`
2. 打开 `http://localhost:3000`

### 日常启动
1. 双击 `start.bat`
2. 打开 `http://localhost:3000`

## V3.2 重点能力
- 案例系统：case / route / site / media_asset / task / evidence / portfolio
- 点位探索：图集预览、图片放大、媒体切换、任务区、AI 问答、证据引用
- 教师工作台：案例列表、新建、编辑、点位编辑、资源管理、任务配置、发布到班级、学生端预览
- 管理员端收缩：仅账号、学校、班级、权限、密码重置

## 内置完整案例
- 黄石国家矿山公园云游（5 个点位）
- 每个点位具备图文、视频、音频、任务与 AI 上下文
